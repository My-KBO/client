# myKBO 프로젝트

### 폴더구조

```
my-app
├── public  # 정적 파일
├─ src  # 컴파일 처리되는 파일
 ├─ components  # 재사용 컴포넌트
 ├─ assets  # 이미지, 폰트 등
 ├─ hooks (= hoc)  # 커스텀 훅
 ├─ pages  # 페이지 컴포넌트
 ├─ constants  # 상수
 ├─ config
 ├─ styles  # 스타일
 ├─ services (= api)
 └── utils  # 유틸리티 파일
├── .gitignore # 깃에 포함x
└── README.md
```
